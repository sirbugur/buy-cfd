Ubuntu/mac için PETSc yükleme:

cd <Enter>  // ana dosya klasörüne gitmek için
git clone -b release https://gitlab.com/petsc/petsc.git petsc
cd petsc

****Bu kısım çok önemli, fortran bindingleri ve dosyalarını compile etmiyoruz, hızlanıyoruz****
./configure --with-cc=gcc --with-cxx=g++ --with-fc=0 --with-fortran-bindings=0 --download-mpich --download-f2cblaslapack

terminalde yazan komutları copy paste
En sonunda, ~/.bashrc ya da profil ne boksa girip,

export PETSC_DIR="/Users/bugur/petsc"
export PETSC_ARCH="arch-darwin-c-debug"

Customize kod yazmak için:
https://docs.petsc.org/en/latest/manual/getting_started/#writing-application-codes-with-petsc
İlgili klasör altındaki Makefile.user kopyalanacak
pkg-config yüklü değilse hata alırsın, brew install pkg-config // sudo apt-get install y pkg-config
